# ~/.dotfiles

Environment setup - macOS/Ubuntu.

## Install

#### macOS

```bash
curl -fsSL https://git.io/ruicdot | bash
```

#### Ubuntu

```bash
wget -qO- https://git.io/ruicdot | bash
```
